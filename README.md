# Amazon Sales & Customer Analysis (SQL Business Case)

# Project Overview
This project analyzes Amazon e-commerce sales data to understand revenue drivers, customer behavior, and product performance using SQL.


#  Business Objective
To derive actionable business insights related to sales trends, customer purchasing behavior, and operational efficiency.


#  Tools Used
- SQL (MySQL)
- CSV Dataset (Amazon Sales Data)


#  Key Analyses Performed
- Total revenue and average order value
- Monthly sales trend analysis
- Product and category performance
- Repeat vs one-time customer behavior
- Top customers by total spend
- Operational cost and order status analysis
